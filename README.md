### Step 1: Install XAMPP

Download and install XAMPP from:
https://www.apachefriends.org

Start the following services in XAMPP Control Panel:

- Apache
- MySQL

---

### Step 2: Copy Project Folder

Copy the project folder into:

C:\xampp\htdocs\

Example:

C:\xampp\htdocs\kitchen_accesories

---

### Step 3: Create Database

1. Open browser
2. Go to:
   http://localhost/phpmyadmin

3. Click **New**
4. Create database with name:

   final

(Important: Database name must be exactly `final`)

---

### Step 4: Import SQL File

1. Click on database **final**
2. Click **Import**
3. Choose file:

   final.sql

4. Click **Go**

Database tables will be created.
